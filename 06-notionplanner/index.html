利用 Notion 制作规划表

本文最后更新于：2023年5月26日 晚上

使用 Notion 独有的简易 Database 来制作规划表

参考自 Thomas Frank 的 Ultimate Tasks for Notion

最终效果如下：

创建 Task Database

在一个规划表里，项目/任务（Task）是一个基本单位。

用户的交互页面只是把所有 Task 进行了归类使得整个操作更加人性化

  1. 在 Notion 里新建一个文件，选择 Table

  2. 此时只是创建了一个新的 Table view，我们还需要到右边的 data source 里新建一个 database

  3. 这样，这份文件才会是 database 的形式

  4. 重命名这份文件，个人来说我会用 “Tasks”

至此，前期准备完毕 ヾ(￣▽￣)

添加 Properties

Checkbox

在每项 item 前面添加一个 checkbox，用于判定此 item 是否完成

在这个 table 的 heading 一栏最右边有个加号 “+”，点击新建 properties

然后选择 checkbox

Property name 实在没有必要，可以删掉留空

Due date

新建一个 Date property，名叫 Due

这样就可以记录这个 item 的截止日期了

创建规划表交互页面

同样，新建一个普通 Page 文件

交互页面应该有三个区域，分别为：

  • Views：这里将所有 Task 按照 due date 分类

  • Projects：项目分类，我这里用的是学校科目

  • Calendar：日历，所有有 due date 的 task 将会在这里按照日历显示

Views

先创建一个 Toggle list（/toggle list），这样如果你不想看到这个列表的话可以收起来

在每个 item 里创建一个 Page（/page），这里我先以 Inbox 为例

来到 Inbox 的 Page，选择 Table

Data source 就选择刚刚创建的 database “Tasks”

现在在 Inbox 的页面应该会显示一个 table，内容就是 “Tasks”
